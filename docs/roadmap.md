# Roadmap

## Backend

- Add request validation with a schema library.
- Add email/SMS notifications for booking status changes.
- Add Razorpay webhook support.
- Add pagination and search indexes for admin lists.
- Add automated API tests.

## Customer Web

- Add payment UI to match the mobile payment flow.
- Add review creation from booking history.
- Add map-based search for supported browsers.

## Customer App

- Add push notifications for booking updates.
- Add native date/time picker for booking schedule.
- Add richer map markers once serviceman geo-coordinates are stored.

## Serviceman App

- Build assigned jobs dashboard.
- Add accept/start/complete booking actions.
- Add availability toggle and service area management.

## Admin Panel

- Build dashboards for users, bookings, revenue and reviews.
- Add serviceman assignment UI.
- Add booking price update workflow.
